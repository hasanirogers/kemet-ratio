# \<kemet-ratio>

## Installation
```bash
npm i @kemet/kemet-ratio
```

## Usage
```js
  import '@kemet/kemet-ratio/kemet-ratio.js';
```

```html
<kemet-ratio>
  <iframe width="100%" height="100%" title="Kemet Templates" src="https://stackblitz.com/edit/kemet-templates?embed=1&file=index.html"></iframe>
</kemet-ratio>
```
